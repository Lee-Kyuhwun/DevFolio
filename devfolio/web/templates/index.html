Portfolio Studio

DevFolio

작업물을 붙여넣고, AI로 구조화하고, 검토한 뒤 이력서와 포트폴리오로 바로 내보내세요.

01
입력

자유롭게 붙여넣거나 직접 입력

02
AI 초안

AI가 구조화된 초안 자동 생성

03
검토

초안 내용 직접 수정·보완

04
내보내기

최종 검토 후 다양한 형식으로 내보내기

처음부터 끝까지, 한 화면에서

붙여넣기에서 문서 생성까지 한 흐름으로

텍스트를 붙여넣는 것부터 문서 내보내기까지 이 화면에서 완료할 수 있습니다.

시작 가이드

처음 시작하는 분을 위한 4단계

아래 단계를 순서대로 완료하면 AI로 포트폴리오를 자동 생성할 수 있습니다.

1

초기 설정 완료

이름과 이메일 등 기본 프로필을 등록합니다. 터미널에서 아래 명령을 실행하세요.

devfolio init
2

AI API 키 발급

AI 초안 생성을 사용하려면 API 키가 필요합니다. 아래 제공자 중 하나를 선택하세요.

Anthropic (권장) OpenAI Google Gemini

키 없이도 수동 입력과 미리보기는 사용할 수 있습니다.

3

AI 키 등록

발급받은 키를 설정 탭에서 등록합니다. OS 키체인에 안전하게 저장됩니다.

4

첫 번째 초안 만들기

입력 탭에서 프로젝트 경험을 붙여넣고 AI로 초안을 생성하거나, Git 저장소를 분석해 자동으로 만들어보세요.

모든 설정이 완료됐습니다. 이제 입력 탭에서 바로 시작해보세요.

Step 1

작업물 붙여넣기

프로젝트 설명, 메모, 회고, PR 정리, 노션 내용 등 편한 형태로 붙여넣으세요.

Step 2-3

초안 검토 및 편집

AI가 만든 초안은 바로 저장되지 않습니다. 수정한 뒤 저장 또는 미리보기를 선택하세요.

Git Scan

저장소 자동 분석

로컬 Git 저장소를 스캔해 내 커밋 기반으로 포트폴리오 프로젝트를 자동 생성합니다.

결과

분석 결과

분석이 완료되면 아래 버튼으로 초안에 바로 불러올 수 있습니다.

분석 결과가 여기에 표시됩니다.

Library

저장된 프로젝트

저장된 프로젝트를 불러와 다시 편집하거나, 저장된 상태에서 AI 문구를 갱신할 수 있습니다.

Selected

프로젝트 관리

에디터로 불러와 수정하거나, AI로 요약·작업 항목을 다시 생성할 수 있습니다.

저장된 프로젝트가 아직 없습니다. Intake 탭에서 초안을 저장하세요.

Step 4

미리보기 및 내보내기

현재 초안이나 저장된 프로젝트를 이력서 또는 포트폴리오 문서로 미리보고 내보낼 수 있습니다.

Selection

저장된 프로젝트 선택

저장된 프로젝트를 소스로 선택하면 어떤 프로젝트를 문서에 포함할지 지정합니다.

아직 생성된 미리보기가 없습니다.

Rendered

문서 미리보기

저장 전 초안도 이 화면에서 바로 미리볼 수 있습니다.

Preview 버튼을 눌러 문서를 생성하세요.

Settings

사용자 프로필

문서 상단에 노출될 기본 정보입니다.

AI

AI 모델 설정

AI를 설정하면 초안 자동 생성과 문구 작성 기능을 사용할 수 있습니다. 수동 작성만으로도 저장과 미리보기는 가능합니다.

Output

내보내기 · 백업 · 일반

문서 기본 포맷과 GitHub 백업 설정을 한 곳에서 관리합니다.
